# `brace-style`

Enforce one true brace style.

## Rationale

Code consistency and maintainability.

## Implementation

```json
{
  "rules": {
    "brace-style": [
      true,
      "1tbs",
      { "allowSingleLine": true }
    ]
  }
}
```

See: https://github.com/buzinas/tslint-eslint-rules/blob/master/src/docs/rules/braceStyleRule.md
