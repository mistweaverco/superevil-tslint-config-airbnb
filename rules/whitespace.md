# `whitespace`

Enforce whitespace style conventions.

## Rationale

Helps maintain a readable, consistent style in your codebase.

## Implementation

```json
{
  "rules": {
    whitespace: [
      true,
      "check-branch",
      "check-decl",
      "check-operator",
      "check-preblock",
      "check-separator"
    ]
  }
}
```

See: https://palantir.github.io/tslint/rules/whitespace/
